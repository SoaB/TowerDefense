# Tower Defense 遊戲專案

這是一個使用 Go 語言和 Ebiten 遊戲引擎開發的塔防遊戲專案。專案旨在提供一個基礎的塔防遊戲框架，包含地圖生成、敵人生成與尋路、防禦塔放置與攻擊等核心功能。

## 功能特色

*   **隨機地圖生成**：每次遊戲開始都會生成獨特的地形地圖。
*   **敵人生成與尋路**：敵人會從地圖邊緣生成，並沿著最佳路徑移動至城堡。
*   **防禦塔系統**：
    *   支援多種防禦塔類型（基礎塔、狙擊塔、急速塔）。
    *   防禦塔會自動攻擊範圍內最近的敵人。
    *   具備攻擊冷卻機制。
*   **攝影機控制**：支援鍵盤移動 (W, A, S, D) 和滑鼠滾輪縮放。
*   **互動式塔放置**：玩家可透過滑鼠點擊在地圖上放置防禦塔。

## 遊戲操作

*   **移動攝影機**：使用 `W`, `A`, `S`, `D` 鍵。
*   **縮放地圖**：使用滑鼠滾輪。
*   **放置基礎塔**：點擊滑鼠左鍵。
*   **放置狙擊塔**：點擊滑鼠右鍵 (測試功能)。

## 開發指南

### 專案結構

主要檔案說明：

*   `main.go`: 遊戲主入口點。
*   `game.go`: 遊戲主邏輯，包含遊戲狀態、更新和繪製。
*   `constant.go`: 定義遊戲中的常數，如地形類型、遊戲狀態、地圖尺寸等。
*   `map.go`: 處理遊戲地圖的生成和管理。
*   `pathfinding.go`: 實作敵人的尋路邏輯。
*   `enemy.go`: 定義敵人結構、行為和管理。
*   `tower.go`: 定義防禦塔結構、行為和管理。
*   `renderer.go`: 負責遊戲中所有元素的渲染。
*   `camera.go`: 處理遊戲攝影機的移動和縮放。

### 如何運行

1.  **安裝 Go 語言環境**：請確保您的系統已安裝 Go 1.22 或更高版本。
2.  **克隆專案**：
    ```bash
    git clone https://github.com/Loowootoo/tower-defense-manus.git
    cd tower-defense-manus
    ```
3.  **安裝依賴**：
    ```bash
    go mod tidy
    ```
4.  **運行遊戲**：
    ```bash
    go run .
    ```

### 如何編譯

```bash
go build -o game_test .
```

這將會在專案根目錄下生成一個名為 `game_test` 的可執行檔。
